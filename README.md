## About Me

I am working on AI agents, scientific machine learning, and reliable model-driven workflows.  
My current interest is building systems that can connect **paper reading**, **code editing**, **experiment execution**, and **iterative improvement** into one loop.

---

## Research Focus

- AI agents for prompting, reflection, code modification, and experiment automation
- Distillation and supervised fine-tuning for coding agents
- Reinforcement learning for agent systems
- Scientific super-resolution and trust-aware prediction
- Long-horizon and multi-scale spatiotemporal / wavefield forecasting

## Selected Projects

### [RL_for_Agent](https://github.com/lkun45598-lgtm/RL_for_Agent)
